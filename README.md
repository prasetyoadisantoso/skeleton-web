# Laravel Skeleton Web
Simple Skeleton to Build Web App

</br>

# Feature
- Multi-language
- Blog
- SEO
- Email
- Settings
- Users & Permissions
- Activity Logging
- Maintenance
- Error Monitoring
- AntiSpam

</br>

# Requirement
## Core
- PHP 8^ and Extensions
- Composer 
- Web Server Apache / NGINX
- MySQL
